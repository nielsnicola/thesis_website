# Read Carefully!

As you can see in this folder there are a few files:

```
footnote.js
index.html
main.css # DO NOT TOUCH!
style.css
```

`footnote.js` makes all footnotes appear as you read, instead of a list at the bottom of the document. If you wish to have the latter and have true "endnotes" simply remove `footnote.js` and remove the following line from the index.html file: `<script src="footnote.js" defer></script>`.

For custom styles, please edit your own styles at the bottom of `style.css` up to your liking. Remember to not go too crazy as this was heavily fraud upon from some of last years theses.

Feel free to edit `index.html` as much as you want, espescially if things did not translate correctly from `.docx` to `.html`, since that was an automated process.

If you want to replace certain images with videos, or add other media files, please do so by editing `index.html` to proparly link to the files, and keep all the files contained within the `media` folder in this folder.

When you're done, simple give back the whole folder!

Goodluck ya'll :D
